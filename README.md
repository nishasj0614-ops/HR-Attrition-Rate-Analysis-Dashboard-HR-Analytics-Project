📊 HR Attrition Rate Analysis Dashboard
📌 Project Overview

The HR Attrition Rate Analysis Dashboard is an interactive analytics project designed to analyze employee turnover and workforce trends. This dashboard provides valuable insights into attrition rate, employee demographics, salary distribution, department-wise analysis, and overall workforce statistics to support data-driven HR decision-making.

🎯 Project Objective

Analyze employee attrition trends

Identify key factors influencing turnover

Monitor workforce demographics

Provide actionable insights for employee retention strategies

📈 Key Metrics Displayed

👥 Total Employees

⚠️ Total Attrition

📊 Attrition Rate (%)

🎂 Average Age

🚺 Female Employees

🚹 Male Employees

Department-wise attrition analysis
